# Tripleten web_project_api_full
Un sitio, connectado a un servidor en google cloud, que utiliza su propio REST api, tecnologias Javascript como react.js, node,js, express.js entre otros, y que fue creado como proyecto final para el curso TripleTen. Contiene funcionalidad de registro/inicio de sesion, agregar tarjetas, eliminar tarjetas creada por el mismo usuario que inicio sesion, seguridad implementada, agregar y borrar likes entre otros. 

# Tecnologias usadas

HTML, CSS, Javascript, React.js, Node.js, Express.js.
